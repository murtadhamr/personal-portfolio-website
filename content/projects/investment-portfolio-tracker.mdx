## Overview

A comprehensive investment tracking tool that aggregates portfolio data from multiple sources, providing real-time analytics and performance metrics.

### Features

- Real-time price updates via WebSocket connections
- Multi-asset class support (equities, bonds, crypto)
- Performance attribution analysis
- Risk metrics dashboard (VaR, Sharpe, drawdown)
- Automated daily reports via email
