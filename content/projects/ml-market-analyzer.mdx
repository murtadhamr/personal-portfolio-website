## Overview

An ML-based market analysis platform that combines traditional time series models with NLP-driven sentiment analysis for comprehensive market insights.

### Features

- Ensemble prediction models (Random Forest, XGBoost, LSTM)
- News sentiment scoring via transformer models
- Backtesting engine with walk-forward optimization
- Interactive visualization dashboard
- REST API for model predictions
